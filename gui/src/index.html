Step 4

Recovery scan

ZECK persists a local wallet workspace while it syncs compact blocks and transparent UTXOs. Cancelling a session keeps that on-disk state for later reuse.
Safe to interrupt. Progress is saved automatically — quit ZECK or cancel the scan at any time, then re-run with the same data directory, network, birthday, and account-scan settings (the same gap limit, or the same explicit account count) to resume from the last persisted block. Changing any of those intentionally starts a fresh scan from the new birthday.
